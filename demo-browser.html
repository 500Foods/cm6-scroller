CodeMirror Custom Scrollbars

Browser-compatible ESM demo

🚀 Browser ESM Usage

This demo loads CodeMirror and the custom scrollbars plugin directly from ESM CDNs. No build process required!

Key imports:

  • import { EditorView, basicSetup } from 'https://esm.sh/@codemirror/basic-setup@6'
  • import { customScrollbars } from './codemirror-custom-scrollbars-browser.js'
JavaScript Editor (Basic Config)
CSS Editor (Custom Config)

Usage Examples

// Basic usage
import { customScrollbars } from 'https://esm.sh/codemirror-custom-scrollbars'

const extensions = [
  basicSetup,
  customScrollbars() // Uses default configuration
]

// Advanced configuration
const extensions = [
  basicSetup,
  customScrollbars({
    scrollbarSize: 14,
    thumbSize: 10,
    thumbBackground: '#007bff',
    thumbHoverBackground: '#0056b3',
    transitionDuration: '0.2s'
  })
]

// CSS variable theming
:root {
  --cm-scrollbar-track-bg: #f0f0f0;
  --cm-scrollbar-thumb-bg: #c0c0c0;
  --cm-scrollbar-thumb-hover-bg: #a0a0a0;
}
